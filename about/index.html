About

Who Am I?

This is me. I'm a highschool student from Baltimore, Maryland, who loves to code. I plan on making it my career after school.

I learned how to code when I was nine, and my first program was a python script where the computer picked a number between 1 and 1000, and you have to guess it. That progressed into more simple python games, like training your multiplication table or finding the n-th Fibonacci number. I then learned HTML, CSS, and JavaScript and made in-browser games.

I'm an ENTP and a Gryffindor. Favorite book is 1984 by George Orwell. Favorite movie is Good Will Hunting.

This Site

This site is made with HTML (obviously). Styling is handled with Boostrap 5, along with my own CSS styles. Many of the projects on this site are created via the p5 Javascript library and HTML5 canvas. Icons are from fontawesome.

This site was hugely inspired by Neal Agrawal, who inititally inspired me to create a website, and Bryan Braun, who inspired me stylistically and conceptually.
